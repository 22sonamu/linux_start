# 사용자
**사람이 아닐 수도 있다**

### :dart: 사용자 확인

```
$ more /etc/passwd
```

### :dart: 파일 소유자 변경
**root사용자(sudo)만 변경할 수 있음**

```
$ chown <변경할소유자이름> <filename>
```
> >*옵션에 -R을 붙히면 하위 디렉토리까지 변경*


# 사용자 그룹
**모든 사용자는 하나 이상의 그룹에 속해 있다**
### :dart: 사용자 그룹 확인

```
$ more /etc/group
```

### :dart: 파일 그룹 소유자 변경
**파일 소유자나 root사용자(sudo)만 변경할 수 있음**
```
$ chgrp <변경할그룹이름> <filename>
```
> >*옵션에 -R을 붙히면 하위 디렉토리까지 변경*


# 프로세스

*2. 시스템 프로세스*
*3. 데몬 프로세스*

```

```

